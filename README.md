# BitmapSimpleResize
画像ファイルを手軽にリサイズ等の加工ができる<br>

## 現在の機能
・画像の縦・横のピクセルを指定して画像サイズを変更<br>
・拡張子を指定して変換<br>

## 今後気が向けば
・テスト(自分で使うだけの予定なので細かいところまで見ていない)<br>
・コードリファクタリングとコメントの追加修正<br>
・UI/UXの改善<br>
・Cドライブ以外の指定

## 追加したい機能
・トリミング機能<br>
・画像サイズをパーセントで変更<br>
・画像を複数読み込んで一括変換<br>

## ※注意
・現在出力先はCドライブ内のパスに固定<br>
・設定の外出しはなし。exeファイルだけで動くようにしている。
